# Inventory Search Application

A simple full-stack application that allows users to search inventory items using filters like product name, category, and price range.

---

## 🚀 Tech Stack

### Frontend
- HTML
- CSS
- JavaScript (Vanilla JS)

### Backend
- Node.js
- Express.js

### Database
- SQLite3

---

## ✨ Features

- 🔍 Search products by name (partial & case-insensitive)
- 🗂️ Filter by category
- 💰 Filter by price range (min & max)
- 🔗 Combine multiple filters
- ⚠️ Handles invalid price ranges
- ❌ Displays "No results found" when no matches
- 📊 Displays results in a clean table UI
